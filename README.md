# Codexa Portfolio

Modern React + Tailwind portfolio for Codexa.

## Stack

- React
- Vite
- Tailwind CSS

## Pages

- `/`
- `/services`
- `/projects`
- `/contact`

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

## Vercel

This repo is ready for Vercel deployment.

- Build command: `npm run build`
- Output directory: `dist`
- SPA rewrite: all routes should resolve to `index.html`
